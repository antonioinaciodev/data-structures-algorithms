# 📚 Data Structures & Algorithms

## 📌 Sobre o Projeto
Este repositório é uma biblioteca pessoal e educacional contendo a implementação das principais **Estruturas de Dados** e **Algoritmos** fundamentais da Ciência da Computação. O objetivo deste projeto é servir como material de estudo, consulta rápida e demonstração de boas práticas de código na resolução de problemas clássicos.

Cada estrutura e algoritmo foi escrito com foco em legibilidade, eficiência e otimização de complexidade de tempo e espaço (Big O Notation).

## 🗂️ Estrutura do Repositório
O projeto está desenvolvido em **Python** e organizado de forma semântica por tipos de dados abstratos e categorias de algoritmos:

*   **`adt/`** (Abstract Data Types):
    *   `bag.py`, `map.py`, `queue.py`, `set.py`, `stack.py`
*   **`linked_lists/`** (Listas Encadeadas):
    *   `singly_linked_lists.py`, `doubly_linked_lists.py`
*   **`priority_queues/`** (Filas de Prioridade):
    *   `array_priority_queue.py`, `linked_priority_queue.py`
*   **`symbol_tables/`** (Tabelas de Símbolos / Dicionários):
    *   `binary_search_symbol_table.py`, `hash_table.py`, `sequential_symbol_table.py`
*   **`trees/`** (Árvores):
    *   `binary_search_tree.py`
*   **`sort/`** (Algoritmos de Ordenação):
    *   `bubble_sort.py`, `selection_sort.py`, `insertion_sort.py`, `shell_sort.py`, `merge_sort.py`, `quick_sort.py`, `heap_sort.py`
