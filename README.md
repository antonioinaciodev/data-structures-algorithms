# 📚 Data Structures & Algorithms

## 📌 Sobre o Projeto
Este repositório é uma biblioteca pessoal e educacional contendo a implementação das principais **Estruturas de Dados** e **Algoritmos** fundamentais da Ciência da Computação. O objetivo deste projeto é servir como material de estudo, consulta rápida e demonstração de boas práticas de código (Clean Code) na resolução de problemas clássicos.

Cada estrutura e algoritmo foi escrito com foco em legibilidade, eficiência e otimização de complexidade de tempo e espaço (Big O Notation).

## 🗂️ Estrutura do Repositório
O projeto está organizado de forma semântica para facilitar a navegação entre os conceitos:

*   **`data-structures/`**: Contém a implementação de estruturas de dados lineares e não lineares.
    *   `Arrays/` & `Strings/`
    *   `LinkedLists/` (Singly, Doubly, Circular)
    *   `Stacks/` & `Queues/`
    *   `Trees/` (Binary Search Tree, AVL, Trie)
    *   `Graphs/` (Adjacency Matrix, Adjacency List)
    *   `HashTables/`
*   **`algorithms/`**: Agrupa os algoritmos por categoria e estratégia de resolução.
    *   `Sorting/` (MergeSort, QuickSort, BubbleSort, etc.)
    *   `Searching/` (Binary Search, Depth-First Search, Breadth-First Search)
    *   `DynamicProgramming/` (Memoization e Tabulation)
    *   `Greedy/`
*   **`challenges/`**: (Opcional) Soluções para problemas famosos de plataformas como LeetCode e HackerRank, utilizando as estruturas acima.

## 🛠️ Tecnologias Utilizadas
*   **Linguagem Principal:** Java (ou adicione a sua linguagem aqui)
*   **Conceitos Focados:** Programação Orientada a Objetos (POO), Design Patterns, Complexidade de Algoritmos (Time/Space Complexity).
